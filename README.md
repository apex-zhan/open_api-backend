# API开放平台后端

> 作者：
## 简介:
一个 API 接口提供开发者调用的平台，基于 SpringBoot 后端 + React 前端项目
GitHub 仓库:
## 实现功能
### 管理员权限:

● 接入并发布接口

● 可视化各接口调用情况

● 发布配置限流、熔断规则 (扩展点)
### 用户权限:
● 开通接口调用权限

● 浏览接口及在线调试

● 通过客户端 SDK 轻松调用接口

### 涉及技术栈:
● 后端：Java, Spring Boot, Spring Cloud Gateway, Dubbo, MyBatis, Nacos

● 前端：Ant Design Pro, ECharts

● 工具：Maven, Git, Nacos, API 签名认证, 限流熔断
